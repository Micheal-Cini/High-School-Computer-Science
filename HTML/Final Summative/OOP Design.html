Object Oriented Programming

Bank Account

The bank account project was the first project that we used Object Oriented Programing for and it taught us many of the basics such as:

Getters & Setters
Constructors
toStrings

These basic concepts were the core ideas that allowed us to properly use Object Oriented Programing

Rectangle

The second project we were assigned using Object Oriented Programing was to calculate the area and perimeter of a rectangle when the user input the length and width

This project used the same concepts as listed in the Bank Account Assignment although the actual code was much simpler.


Temperature Conversion

The third and final Object Oriented Programming project we did was to convert temperature between celsius and fahrenheit

Similar to the other projects, the same concepts are used to demonstrate OOP design.
