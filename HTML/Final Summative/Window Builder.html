Window Builder

Trivia Game

For this project, we had to use Window Builder to design a GUI that would perform a task. In my case, I created a trivia game that gives random trivia questions with 4 answers. I did this by useing text files to store the questions and answers, and insert them into arrayLists. I then seperated the arrayLists into strings to be placed on the answer buttons.




Text Escape Game

In this project, I made a text adventure game that required the user to complete a set of puzzles to advance through the game. I submitted this project for my grade 11 final summative. At the time I was very happy with the code, but now, knowing how I can make it bettter, I hope to remake it one day with a better design. I have many ideas of how I can improve on the story as well as the actual code behind the game.
