Data Structures

Address Book

In this assignment, we had to create an address book capable of storing names, addresses, and phone numbers specified by the user. I think that out of all of the projects, this one was my favourite. It introduced 2 new concpets called ArrayLists, and inheritance. This project is what led me to use ArrayLists for pretty much any project done after this.

Searching & Sorting

For this assignemnt, we introduced the new concpet of reading and writing from text files, as well as implementing several different searching and sorting methods such as binary search, linear search, bubble sorting, and insertion sorting. These new ideas allowed us to make much more efficent and reliable code for use in future projects.

String Builder

This concept was not really a project. It was more of an example done in class to show off the "String Builder" concept. String builder allows us to take strings and replace, insert, or delete certain parts of the code with another string. To be honest I haven't used string builder much in my projects but it is definately a useful tool.
